# Tutorial 01: Simple Game

This directory contains the simple interactive game from Tutorial 1.

## Running the Game

```bash
# Make sure you're in the root (main) directory of your project
uv run src/tutorial_01_simple_game/simple_game.py 
```

## Controls

- **Arrow Keys**: Move the circle up, down, left, right
- **Close Window**: Exit the game

## What This Game Demonstrates

- Basic Pygame initialization
- Creating a game window
- The game loop pattern
- Event handling (keyboard input and window close)
- Drawing shapes on screen
- Frame rate control with clock
